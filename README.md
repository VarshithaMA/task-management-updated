👤 Person 1 — About the Project

Our project is Task Management System.

The main goal of this system is to help users manage and organize tasks efficiently.

Users can:

Create tasks

Update tasks

Delete tasks

Assign tasks to employees

Track task progress

The system also provides user login, profile management, and notifications.

This helps improve productivity and work organization.

👤 Person 2 — Tech Stack

This project is developed using the following technologies:

Frontend:

HTML

CSS

Bootstrap

Backend:

PHP

Database:

MySQL

Server:

XAMPP (Apache Server)

These technologies help build a dynamic web application where users can manage tasks efficiently.

👤 Person 3 — Frontend

The frontend is designed using HTML, CSS, and Bootstrap.

It provides a simple and user-friendly interface.

Users can:

Login to the system

View tasks

Create new tasks

Edit or delete tasks

View notifications

Manage their profile

Bootstrap helps in creating a responsive design so the system works on different screen sizes.

👤 Person 4 — Backend

The backend is developed using PHP.

It handles:

User authentication

Task creation

Task updates

Task deletion

Database communication

The backend connects with the MySQL database using DB_connection.php.

All the task data and user information are stored securely in the database.

👤 Person 5 — Real World Use

This system is useful in many real-world scenarios.

In companies:

Managers can assign tasks to employees

Employees can track their assigned work

Teams can monitor project progress

Apart from companies, it can also be used in:

Colleges for managing assignments

Event management teams

Freelancers managing client work

Small businesses organizing daily tasks

This system helps improve productivity, organization, and time management.
